# CS 312 - SDG 11 Static Page

## Overview

This project is a static webpage created as part of the prelim term of the CS 312 course. The page is designed using HTML5 and CSS3 and focuses on Sustainable Development Goal (SDG) 11: Sustainable Cities and Communities.

## Project Purpose

The primary goal of this project is to provide a practical exercise in web design and development by creating a static page that highlights SDG 11. This includes applying modern HTML5 and CSS3 techniques to build a visually appealing and informative webpage.

## Features

- **HTML5**: Semantic elements and structure
- **CSS3**: Styling and layout design
- **Focus**: Sustainable Cities and Communities (SDG 11)

## Getting Started

To view this project, you can:

1. **Clone the Repository**:
    ```bash
    git clone <repository-url>
    ```

2. **Navigate to the Project Directory**:
    ```bash
    cd <project-directory>
    ```

3. **Open the Static Page**:
    Open `index.html` in your preferred web browser to view the static page.
